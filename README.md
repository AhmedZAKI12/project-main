# Project Main

A frontend web application built using React, Vite, and Tailwind CSS.

## Features
- Modern React setup
- Fast development with Vite
- Responsive UI using Tailwind CSS

## Tech Stack
- React
- Vite
- Tailwind CSS

## Installation
npm install
npm run dev
